# Moving America: Interstate Migration and Labor Market Change

This project is an interactive Streamlit dashboard that explores interstate migration patterns across U.S. states.

## Features
- Migration map showing net migration by state
- State-level trends over time
- Migration vs unemployment relationship
- Migration network visualization

## Data
The dataset includes cleaned interstate migration flows and state-level indicators.

## How to run locally
streamlit run app.py
